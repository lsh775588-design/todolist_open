Streamlit 기반 To-Do List 웹 애플리케이션

주요 기능
1. 할 일 추가 및 자동 저장
2. 엔터키 입력 지운 : 버튼을 클릭하지 않고 새롭게 추가한 할일을 엔터키로 등록할 수 있다
3. 입력창 초기화 : 등록이 성공하면 입력칸이 비워진다
4. 피드백 알림 : 성공 및 공백 예외 처리 경고 메세지를 출력한다
5. 개별 항목 삭제 : 각 항목 우측에 있는 X 표시를 사용하여 항목을 삭제할 수 있다
6. 전체 초기화 : 전체 삭제 버튼을 활용하여 모든 항목을 삭제할 수 있다


변경 사항 
1. 기존의 클릭 기반에서 엔터키 사용을 추가
2. 사용자가 빈 항목을 추가를 시도할때 예외처리를 추가
3. st.columns를 활용하여 레이아웃을 쪼개고 개별 삭제 기능을 추가

실행 방법
1. 필수 라이브러리 : pip install streamlit
2. 터미널에 입력 : streamlit run 프로젝트파일명.py
